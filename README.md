# SmallMatPrep (v3_python)

A minimal Python library for small-sample materials data diagnostics, imputation, model recommendation, and evaluation.

## Quick start

```bash
cd v3_python
pip install -e .
python examples\run_minimal_demo.py
```

## Core workflow
1. Load sample data and config
2. Inspect missingness
3. Apply imputation
4. Get model recommendations
5. Evaluate results
6. Print summary
